Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Podemos criar um texto grande usando a tag BIG (obsoleto).

Podemos criar um texto pequeno usando a tag SMALL.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele dever ser lido, mas não considerado, usando a tag DEL.

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois, usando a tag INS (semântica).

Podemos marcar um texto como sublinhado usando a tag U (não semântica).

Texto subescrito

Podemos marcar um texto como subescrito H2O usando a tag SUB.

Texto sobrescrito

Podemos marcar um texto como sobrescrito x20+3 usando a tag SUP.
