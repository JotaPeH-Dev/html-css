Lista ordenada

Podemos criar uma lista ordenada usando a tag OL (Ordered List) e cada item da lista é criado com a tag LI (List Item). São listas onde a ordem dos itens importa.

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

A tag "ol" possui parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esses parâmetro são:

  • type="1" - Marcador numérico (padrão)
  • type="A" - Marcador alfabético maiúsculo
  • type="a" - Marcador alfabético minúsculo
  • type="I" - Marcador romano maiúsculo
  • type="i" - Marcador romano minúsculo

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Podemos também pode indiciar o início da contagem usando o parâmetro start.

Por exemplo, a tag "<"ol" type="I" start="5">"" vai gerar uma lista com início em V, VI, VII, VIII, IX,...

Lista não ordenada

Podemos criar uma lista não ordenada usando a tag UL (Unordered List) e cada item da lista é criado com a tag LI (List Item).

  • Ovo
  • Leite
  • Tomate
  • Alface

A tag "ul" possui parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esses parâmetro são:

  • type="disc" - Marcador em forma de disco (padrão)
  • type="circle" - Marcador em forma de círculo
  • type="square" - Marcador em forma de quadrado
